Instructions d'utilisation
1. Générer les données
Exécuter python generate_traffic_data.py
dans le dossier data_generator/
2. Extraire vers HDFS
Exécuter python extract_to_hdfs.py
dans le dossier extraction/
3. Lancer les travaux
Cliquer sur Lancer tous les travaux ou exécuter manuellement
bash run_jobs.sh all
4. Visualiser
Le tableau de bord se met à jour automatiquement après chaque travail. Cliquer sur Actualiser pour forcer le rechargement.
